# 狗狗寄养中心房间管理小程序

这是一个为狗狗寄养中心设计的房间管理微信小程序，功能类似酒店房态管理系统。

## 🎯 核心功能

### 1. **房间管理**
- 自动初始化60个房间：28间标准庭院房(98元/天)、24间豪华庭院房(128元/天)、8间贵宾房(198元/天)
- 实时显示房间状态：空闲(绿色) / 已占用(红色)
- 支持添加、编辑、删除自定义房间

### 2. **狗狗档案管理**
- 完整记录：名字、性别、品种、年龄、健康状况(咬人/咬狗/疫苗/绝育/驱虫)
- 主人信息：姓名、电话、特殊要求
- 自动标记狗狗当前状态：在店/离店
- 支持搜索功能

### 3. **智能预约系统**
- **防冲突检查**：自动检测房间在选定时间段是否可用
- 可视化选择：根据房间类型、时间筛选可用房间
- 一键预约：选择狗狗→选择时间→选择房间→确认
- 预约管理：查看、取消当前预约

### 4. **每日视图**
- 按日期查看：哪些房间有狗、入住/离店时间
- 清晰展示：房号 → 狗狗名字 → 寄养时间段
- 今日统计：入住数、离店数、在店狗狗数

### 5. **记录查询**
- 多维度搜索：按狗狗名字、主人电话、房间号
- 时间线展示：所有寄养历史记录
- 分类查看：全部记录/狗狗记录/房间记录

### 6. **仪表盘**
- 系统概览：总房间数、登记狗狗数、已占用房间、今日入住
- 房间状态分布可视化
- 今日提醒：入住、离店、在店狗狗等信息
- 快速操作入口

## 📱 界面特点
- **移动端优先**：响应式设计，完美适配手机
- **黑白极简风**：专业、清晰、不刺眼
- **操作高效**：大按钮、简洁表单、快速导航
- **视觉反馈**：实时状态更新、操作成功提示

## 🚀 使用说明

### 首次使用
1. 使用微信开发者工具导入本项目
2. 编译运行后，系统会自动初始化60个标准房间
3. 从"狗狗档案"页面开始添加狗狗信息
4. 从"寄养预约"页面创建第一个预约

### 日常操作流程
1. **添加狗狗档案**：在"狗狗档案"页面添加新狗狗信息
2. **创建预约**：在"寄养预约"页面选择狗狗、时间、房间
3. **查看状态**：通过"每日视图"查看当天房间占用情况
4. **管理房间**：在"房间管理"页面编辑房间信息
5. **查询记录**：在"记录查询"页面搜索历史记录

### 数据存储
- 所有数据保存在微信小程序的本地存储中
- 数据在卸载小程序前会一直保留
- 支持多人同时使用（不同设备数据独立）

## 🔧 技术特点
- **纯前端实现**：WXML + WXSS + JavaScript，无需后端
- **本地存储**：使用微信小程序Storage API，数据持久化
- **离线可用**：无需网络连接即可使用
- **跨平台**：支持所有微信版本

## 💡 注意事项
1. 首次运行会自动初始化房间数据
2. 数据保存在本地，请勿随意清除小程序数据
3. 时间选择支持日期时间选择器
4. 所有操作都有确认提示，防止误操作
